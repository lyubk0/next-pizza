# 🍕 Next Pizza

A modern full-stack pizza ordering web application built with Next.js.

The application allows users to browse pizzas, customize their orders by selecting sizes and ingredients, add items to the cart, and complete purchases through an integrated payment system.
The project focuses on building a scalable e-commerce architecture with clean UI components and efficient state management.

# ⚙️ Getting Started

### 1️⃣ Clone the repository

```bash
git clone https://github.com/lyubk0/next-pizza
```

### 2️⃣ Navigate to the project

```bash
cd next-pizza
```

### 3️⃣ Install dependencies

```bash
npm install
```

or

```bash
pnpm install
```

### 4️⃣ Run the development server

```bash
npm run dev
```

Open in browser:

```
http://localhost:3000
```
